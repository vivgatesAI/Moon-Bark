# MoonBark

MoonBark is a beautiful Venice API web app for recording a dog's voice and turning it into a conversational bridge between human and dog.

## Features

- record dog audio in-browser
- live waveform visualization
- bark-energy / bark-count inference from audio features
- Venice chat-powered dog intent interpretation
- Venice text-to-speech dog reply playback
- human-to-dog conversation mode
- lush, gallery-like UI

## Run

```bash
npm install
npm start
```

Then open `http://localhost:4328`.

## Environment

Requires `VENICE_API_KEY` in the environment.

## Notes

This is an imaginative conversation agent, not a scientific canine translation system. It uses acoustic features, context, and Venice language generation to produce emotionally plausible dog-side interpretations and replies.
